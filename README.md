## 📖 О проекте

**Оператор САПЁР** — восьмой урок в серии бесплатных обучающих тренажёров для подготовки операторов БПЛА. Приложение моделирует процесс обнаружения и обезвреживания мин с помощью специализированного дрона-сапёра, оснащённого металлоискателем.

Тренажёр реалистично симулирует управление квадрокоптером в 3D-пространстве, включая физику движения, инерцию, высоту полёта и систему обнаружения взрывоопасных предметов с помощью звуковых и визуальных индикаторов.

## ✨ Возможности

- 🎮 **Реалистичная физика дрона** — инерция, сопротивление воздуха, наклоны при движении
- 🔍 **Металлоискатель** — звуковой и визуальный индикатор интенсивности сигнала
- 🧭 **Компас** — красная стрелка указывает направление к ближайшей мине
- 🎯 **Три уровня сложности** — от новичка до профессионального сапёра
- 🏆 **Система рекордов** — сохранение лучшего времени для каждого уровня (localStorage)
- 📱 **Мобильное управление** — виртуальный джойстик и сенсорные кнопки
- 🎵 **Звуковые эффекты** — детектор, обезвреживание, взрывы, фоновая музыка
- 🌟 **Красивая 3D-графика** — тени, частицы, эффекты взрывов, звёзды
- ⚡ **Оптимизация** — автоматическое определение слабых устройств
- 💥 **Реалистичные эффекты** — тряска камеры при столкновениях, вспышки взрывов
- ⏸️ **Пауза** — возможность остановить игру в любой момент

## 🎮 Уровни сложности

## 🕹️ Управление

### Клавиатура и мышь (ПК)
- `W/↑/Ц` — Вперёд
- `S/↓/Ы` — Назад
- `A/←/Ф` — Влево
- `D/→/В` — Вправо
- `Q/Shift` — Набор высоты ⬆️
- `E/Ctrl` — Снижение ⬇️
- `ЛКМ` (удерживать) — Обезвреживание мины 🎯
- `ESC` — Пауза ⏸️

### Мобильные устройства
- 🕹️ **Левый джойстик** — горизонтальное движение
- ⬆️ **Кнопка вверх** — набор высоты
- ⬇️ **Кнопка вниз** — снижение
- 🎯 **Красная кнопка** — обезвреживание (удерживать)
- ⏸️ **Кнопка паузы** — пауза

## 🎯 Цель игры

1. Найдите все мины на территории, ориентируясь на сигнал металлоискателя и компас
2. Подлетите к мине и удерживайте её в зоне обезвреживания (зелёное кольцо)
3. Удерживайте кнопку обезвреживания 2 секунды, пока мина не будет ликвидирована
4. Обезвредьте все мины до истечения времени
5. Не допускайте столкновений с препятствиями и слишком жёстких посадок!

## 🚀 Запуск

### Быстрый старт
```bash
# Клонируйте репозиторий
git clone https://github.com/YOUR_USERNAME/saper-drone-simulator.git

# Перейдите в папку проекта
cd saper-drone-simulator

# Откройте index.html в браузере
# Или запустите локальный сервер:
python -m http.server 8000
# Затем откройте http://localhost:8000
```

### Требования
- Современный браузер с поддержкой WebGL (Chrome, Firefox, Edge, Safari)
- Включённое аппаратное ускорение в настройках браузера
- Для мобильных устройств: сенсорный экран

## 🛠️ Технологии

- **HTML5** — структура и семантика
- **CSS3** — стили, анимации, адаптивность

## ⚙️ Конфигурация

Основные параметры можно настроить в объекте `CONFIG` внутри `<script>`:

```javascript
const CONFIG = {
  DRONE_SPEED: 0.06,        // Скорость дрона
  MAX_ALTITUDE: 40,         // Максимальная высота
  MIN_ALTITUDE: 2,          // Минимальная высота
  CAPTURE_TIME: 2000,       // Время обезвреживания (мс)
  DETECTOR_RANGE: 10,       // Дальность детектора
  // ... и другие параметры
};
```
